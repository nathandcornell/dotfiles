# Nate's dotfiles

## Installation
1. Clone this repo
2. Pull submodules: `git submodule update --init --recursive`
3. Run dependencies.sh to install homebrew, git, and other core dependencies
4. Run setup.sh
5. Create ~/.gitconfig-user.inc using the example
